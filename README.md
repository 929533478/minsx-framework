# Minsx-Framework
Minsx（咪斯艾克斯 ）是一款基于Spring的轻量级WEB框架，她可以让你以最短的时间开发出企业级应用

###项目简介
+ 主要用于小型云平台后端应用
+ 较为便捷,开箱即用
+ 项目采用前后端分离
+ 我们提供默认通用用户前端实现(请参照：minsx-user-ui)
+ 我们提供默认通用管理前端实现(请参照：minsx-management-ui)
+ 采用Restful-URL进行开发
+ 代码规范方面遵循最新阿里巴巴JAVA代码规范
